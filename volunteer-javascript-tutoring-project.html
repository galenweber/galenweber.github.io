Volunteer JavaScript tutoring project

Through April 2022, I'll be offering tutoring in JavaScript and "JavaScript-based" languages (Node, TypeScript) via chat, recorded video, and VR.

Tutoring is free for anyone who is:

  • Low income (OR)
  • From a group that is underrepresented in tech (OR)
  • Feels they need help right now

I can probably only effectively tutor fluent English speakers (unfortunately, I'm not fluent in any other language).

Details

Process

Just message me on Discord! Join my server using the link below, and DM your question. Stack Overflow has a good guide on writing good questions ↗.

https://discord.gg/6QacvhfbeU

Format

I'll do my best to answer any question you send me, either in writing, a recorded video, or in a "Horizon Workroom" ↗ (see VR JavaScript tutoring for more detail). I'll try to provide an answer that helps you learn.

I think the "value-add" for this type of service is to answer questions that you can't post to Stack Overflow. Questions like: "Why isn't this function passing this test case?".

To start, I won't be offering synchronous tutoring over video.
